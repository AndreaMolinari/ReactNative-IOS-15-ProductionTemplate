# ReactNative

## Target
Creare un template per una applicazione sia mobile che web

```sh
git clone git@github.com:AndreaMolinari/ReactNative-IOS-15-ProductionTemplate.git MyNewApp

cd MyNewApp

npx react-native-rename MyNewApp
```
